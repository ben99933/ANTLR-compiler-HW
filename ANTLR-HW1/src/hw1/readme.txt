+===========================================================+
|   如何編譯:
|   terminal(Linux環境或WSL)中在本目錄之下輸入以下指令:
|   make
+-----------------------------------------------------------+
|
|   如何執行:
|   在terminal(Linux環境或WSL)中在本目錄之下輸入以下其中一個指令:
|       1. make test1
|       或
|       2. make test2
|       或
|       3. make test3
|
|   以上3個指令分別將會先編譯，然後將./test目錄之下的
|   test1.c, test2.c, test3.c 作為input argument執行
|
+-----------------------------------------------------------+
|
|   專案架構
|   ANTLR存放於 ./lib/antlr-4.9.2-complete.jar
|   ./src/目錄存放所有的source code
|   ./test/目錄存放所有的test case
+===========================================================+
